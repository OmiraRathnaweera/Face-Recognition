# Facial Expression Recognition (FER) Using AI & Machine Learning
This repository contains a deep learning project focused on classifying human facial expressions into three distinct emotional states: Disappointed, Neutral, and Interested. The model is trained and validated using the FER_2023 dataset sourced from Kaggle.

## Project Overview
Facial Expression Recognition (FER) is a pivotal component of Affective Computing. While many datasets focus on basic emotions (like Joy or Anger), this project focuses on nuanced states—Disappointed and Interested—alongside a Neutral baseline. This has practical applications in user experience (UX) research, educational monitoring, and customer sentiment analysis.

## Dataset
The model utilizes the FER_2023 dataset from Kaggle.

Source: https://www.kaggle.com/datasets/msambare/fer2013

Labels Used: Disappointed, Neutral, Interested.

Preprocessing: Images were grayscale-normalized and resized for optimal feature extraction through Convolutional Neural Networks (CNNs).
